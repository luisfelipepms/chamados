#Chamados

##Como executar o projeto

Para iniciar a aplicação, é necessário ter o Docker e o Docker Compose instalados na máquina.

No diretório raiz do projeto, execute o comando:

docker-compose up --build

Esse comando irá construir as imagens necessárias e iniciar todos os serviços da aplicação.

Após a inicialização, o sistema estará disponível conforme as portas configuradas no arquivo docker-compose.yml.
